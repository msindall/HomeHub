📅 Connect Google Calendar
🔗 How to get your Live Link (one-time setup):
1. Open Google Calendar in a browser
2. Click the ⚙️ gear icon (top right) → Settings
3. In the left sidebar, click your calendar name (e.g. your name)
4. Scroll down to "Integrate calendar"
5. Copy the Secret address in iCal format link
6. Paste it below — Home Hub will sync automatically every time you visit! ✅
📁 How to download your .ics file:
1. Open Google Calendar in a browser
2. Click ⚙️ Settings → Import & Export
3. Click Export — saves a .zip to your Downloads
4. Unzip it — you'll find one or more .ics files inside
5. Click Choose File below and select the .ics file
📋 How to get raw iCal text:
1. In Google Calendar → ⚙️ Settings → click your calendar → Integrate calendar
2. Copy the Secret address in iCal format link
3. Open that link in a new tab — you'll see text starting with BEGIN:VCALENDAR
4. Press Ctrl+A then Ctrl+C to copy all, then paste below ↓

🔑 Anthropic API Key
AI features (meal plan, flyer scan, PDF statements) require an Anthropic API key — this works in any browser or the Claude Desktop App.

How to get one (free):
1. Go to console.anthropic.com
2. Sign up / log in → click API Keys
3. Click Create Key → copy it
4. Paste it below ↓

Your key is stored locally in your browser and never sent anywhere except Anthropic's API.
