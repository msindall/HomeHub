📋 How to Export Your Statements
🏦 RBC Chequing / Savings
Online Banking → Accounts → select account → Download Transactions → CSV
File: downloadtransactions.csv
💳 TD / Amex Credit Card
EasyWeb → Accounts → select card → Download → choose range → CSV
File: *transaction_download*.csv
🏦 BMO Chequing / Savings
BMO Online Banking → Accounts → select account → Download Transactions → CSV
File: statement BMO chequing.csv (or similar)
🍁 Canadian Tire Mastercard
MyCardInfo → Accounts → select card → Download Transactions → CSV
File: MY ACCOUNT TRANSACTIONS (REF column included)
📄 Any Bank — PDF Statement
Online Banking → Statements → Download PDF. Scanned PDFs use AI Vision (20–40 sec)
Works for RBC, TD, CIBC, BMO, Scotiabank
